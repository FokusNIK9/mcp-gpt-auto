# Local Dev Agent - Firefox Companion

Мощный компаньон для ChatGPT, который автоматизирует рутинные действия при работе с локальным Action Bridge.

## Основные функции

1.  **Accept Mode (Авто-подтверждение):**
    *   Кнопка **"Accept: ON"** автоматически нажимает "Confirm" / "Allow" / "Подтвердить" для вызовов инструментов.
    *   Работает независимо от таймера авто-отправки.
    *   **Безопасность:** Кликает только в диалогах, содержащих ваш ngrok URL или метку "Local Dev Agent".
2.  **Auto Mode (Авто-кликер):**
    *   Кнопка **"Agent Auto: ON"** отправляет сообщения в чат по таймеру (интервал настраивается).
    *   Автоматически очищает поле ввода перед новой отправкой.
    *   Поддержка префиксов: Теги, короткие уникальные ID и время отправки.
3.  **Multi-Sync (Динамические файлы):**
    *   Автоматическое получение и прикрепление одного или нескольких файлов с вашего ПК.
    *   В поле **Bridge Paths** (под шестеренкой) укажите пути через запятую: `text.txt, logs/runner.log`.
    *   Аддон дожидается завершения загрузки файлов перед отправкой сообщения.
4.  **Умная проверка:**
    *   Аддон ждет завершения генерации ответа ChatGPT (`Status: Waiting for ChatGPT...`).
    *   Можно принудительно пропустить проверку готовности (галочка "Skip Check").

## Установка (Временная)

1.  Откройте Firefox и перейдите на страницу `about:debugging#/runtime/this-firefox`.
2.  Нажмите кнопку **"Load Temporary Add-on..."**.
3.  Выберите файл **`manifest.json`** в папке `addons/firefox-action-confirm/`.
4.  **Важно:** Временные аддоны удаляются при перезапуске браузера.

## Как пользоваться

### Панель управления (справа внизу)
*   **Agent Auto:** Вкл/Выкл таймер отправки сообщений.
*   **Accept:** Вкл/Выкл автоматическое подтверждение MCP-запросов.
*   **Иконка ⚙️:** Расширенные настройки (Теги, Bridge Paths, ID).
*   **Иконка 📎:** Ручной выбор файлов.
*   **Иконка ➤:** Немедленная отправка с текущими настройками.

## Настройка синхронизации
1.  Запустите сервер `mcp-gpt-auto`.
2.  Нажмите ⚙️ и включите **"Sync from Bridge"**.
3.  Впишите путь к файлу (абсолютный или относительный): `C:\путь\к\файлу.md`.
4.  Теперь аддон сам будет «подтягивать» свежую версию файла при каждой отправке.

## Безопасность
Аддон работает только на доменах `chatgpt.com`. Он не собирает данные и общается только с вашим локальным сервером. Скачивание файлов происходит через фоновый скрипт аддона для обхода ограничений безопасности браузера.
